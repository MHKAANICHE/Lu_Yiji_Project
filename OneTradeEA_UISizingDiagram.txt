## Legend
- All X, Y, W, H values are in pixels.
- Each block is shown in its relative position and size.
- Button positions are calculated from ButtonRow X + padding and gaps.

## Usage
- Use this diagram to visually confirm the layout and sizing of all UI elements before coding or adjusting the MQL5 implementation.
- For graphical design, this can be converted to a vector diagram or used as a reference for Figma/Sketch.
